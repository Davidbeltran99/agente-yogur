Extrae un pedido a JSON.

Reglas:
- Responde solo JSON válido
- Si falta un dato, usa null
- No inventes productos
- Copia el nombre del producto como lo escribió el cliente
- Usa varios items en `productos` si hay varios productos
- `estado` siempre es `pendiente`
- Si no parece pedido, devuelve `productos: []`
- Si el cliente menciona varios productos, extrae cada uno por separado
- Cuando aparezca una nueva cantidad, asume que empieza otro item si cambia el producto
- Conserva typo humano, tamaño implícito, intención de barato/económico y referencias como “el mismo de la otra vez” dentro del texto del producto u observaciones
- Conserva cantidades, variantes implícitas (grande/pequeño/barato) y observaciones útiles en `observaciones`
- Si el texto viene de OCR o visión de una imagen, trátalo como un pedido ya leído; no lo descartes por venir de imagen

Estructura:
{
  "cliente": "",
  "productos": [
    {
      "producto": "",
      "sabor": "",
      "cantidad": 0
    }
  ],
  "direccion": "",
  "fecha_entrega": "",
  "metodo_pago": "",
  "observaciones": "",
  "estado": "pendiente"
}
